### About me

I am a software engineer based in Tunisia with a Holberton School foundation and a habit of shipping real products. I build across the stack, from web apps and APIs to command line tools and game server scripts. I also run my own online store selling scripts and assets to a worldwide customer base, which has taught me as much about support, releases, and customers as it has about code.

I care about clean, readable code, solving the actual problem, and finishing what I start.
